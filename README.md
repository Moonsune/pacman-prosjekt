# 🎮 Pacman i Java
Et klassisk Pacman-spill laget med Java som del av et skoleprosjekt

## Funksjoner
- Spillbar Pacmac med bevegelige spøkelser
- Poengsystem
- Livsystem
- Kollisjonsdeteksjon (vegger, fiender, poeng)
